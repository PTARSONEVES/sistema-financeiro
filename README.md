# 💰 Sistema Financeiro - Contas a Pagar e Receber

Sistema completo para gestão financeira pessoal com dark mode, autenticação e dashboard interativo.

## 🚀 Tecnologias

### Backend
- Node.js
- Express
- MySQL
- JWT para autenticação
- Bcrypt para criptografia

### Frontend
- React
- React Router DOM
- Axios
- Tailwind CSS
- Context API para gerenciamento de estado

## ✨ Funcionalidades

- ✅ Autenticação de usuários (registro/login)
- ✅ Dashboard com visão geral financeira
- ✅ CRUD completo de contas a pagar/receber
- ✅ Filtros por tipo e status
- ✅ Paginação
- ✅ Dark mode 🌓
- ✅ Design responsivo
- ✅ Breadcrumbs de navegação

## 📋 Pré-requisitos

- Node.js (v18 ou superior)
- MySQL (XAMPP, WAMP ou MySQL Server)
- npm ou yarn

## 🔧 Instalação

### Backend
```bash
cd backend
